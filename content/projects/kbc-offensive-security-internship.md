## Context

From February 2026 to May 2026, I completed an offensive security internship at KBC. The work focused on mobile penetration testing, Android banking malware research, and automation for security testing environments.

## Technologies

- Burp Suite
- Frida
- OWASP MASVS
- Android security testing
- Malware analysis tooling
- Automation scripting

## My Contribution

I developed automation for mobile penetration testing environments, integrated Burp Suite with Frida for login flow automation, contributed to a KBC mobile security assessment based on OWASP MASVS, and researched Android banking malware.

## Reflection

This internship gave me practical experience in a real offensive security environment. It connected mobile security theory with day-to-day testing work, where automation, repeatability, and clear reporting matter as much as finding the issue itself.
