## Context

This project built a Burp Suite and ELK Stack setup for centralized storage, analysis, and anomaly detection of HTTP requests.

## Technologies

- Burp Suite
- Elasticsearch
- Logstash / Beats
- Kibana
- Filebeat
- HTTP request analysis

## Result

The setup supports security research and vulnerability tracking by collecting and analyzing HTTP traffic in a centralized monitoring stack.

## Reflection

This project connected penetration testing workflows with monitoring and analysis. It pushed me to think beyond individual findings and look at how request data can be stored, searched, and used during longer security investigations.
