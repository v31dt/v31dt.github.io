How I Built This Portfolio

28 2026

This portfolio is built with Hugo, a static site generator. I chose Hugo because it keeps the portfolio simple, fast, and easy to maintain. Each article is written as a Markdown file, and Hugo converts those files into a static website.

The visual theme is Archie, a minimal Hugo theme. I selected it because it keeps the focus on the content: reflections, projects, and proof of professional development.

The source code is hosted in the GitHub repository v31dt/v31dt.github.io. GitHub Pages publishes the repository from the main branch. The generated Hugo output is committed in the repository root so the current GitHub Pages configuration can serve the website directly.

Professional Networking Checklist

28 2026

This page tracks the required portfolio items for the Professional Networking module.

Required items

  • Portfolio website available through a public URL
  • About page that clearly identifies who owns the portfolio
  • Blog post explaining how the website was created
  • Four physical event reflections with proof of attendance
  • Hackathon reflection with proof of participation
  • Podcast episode with an external IT expert
  • LinkedIn profile URL, PDF export, and reflection
  • TI on STAGE pitch material and reflection
  • Final PDF version of the website for submission
